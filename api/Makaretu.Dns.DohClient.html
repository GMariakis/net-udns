Class DohClient

Client to a DNS server over HTTPS.

Remarks

DNS over HTTPS (DoH) is an experimental protocol for performing remote Domain Name System (DNS) resolution via the HTTPS protocol. The goal
is to increase user privacy and security by preventing eavesdropping and manipulation of DNS data by man-in-the-middle attacks.
