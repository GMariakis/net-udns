DNS Clients

Various DNS clients that communicate with a DNS server. The source code is on GitHub and the package is published on NuGet.

  • DnsClient is used to send a Message to a standard DNS server and receive a response; uses UPD and then TCP (if needed).
  • DohClient is used to send a Message to a DNS over HTTPS server and receive a response.
  • DotClient is used to send a Message to a DNS over TLS server and receive a response.

Usage

Get all the TXT strings assoicated with "ipfs.io"

using Makaretu.Dns;

var dns = new DnsClient();
var response = await dns.QueryAsync("ipfs.io", DnsType.TXT);
var strings = response.Answers
    .OfType<TXTRecord>()
    .SelectMany(txt => txt.Strings);
foreach (var s in strings)
    Console.WriteLine(s);

Produces the output

dnslink=/ipfs/QmYNQJoKGNHTpPxCBPh9KkDpaExgd2duMa3aF6ytMpHdao
